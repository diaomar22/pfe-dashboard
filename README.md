# Dashboard PFE — Prévision de l'IPI mauritanien

Dashboard interactif présentant mon Projet de Fin d'Études en Licence MAEF (I.S.G.I.) :
modélisation et prévision de l'Indice de la Production Industrielle de la Mauritanie
par un modèle SARIMA.

**Stage réalisé à l'ANSADE.**

## Contenu

- **Présentation** : objectif, démarche, résultats
- **Série et prévisions** : graphique interactif IPI 2011–2025 + prévisions 2026
- **Décomposition** : tendance et saisonnalité
- **Tableau des prévisions** : valeurs 2026 avec intervalles de confiance

## Modèle

SARIMA(0,1,2)(0,1,1)₄ · MAPE = 3,77% · RMSE = 4,90
